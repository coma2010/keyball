# keyball44 user1 keymap

更新日: 2025-11-05

- `キー配列`: `keymap_cheatsheet_yowkees_keyball44.pdf` (2025-11-04 版) と揃えました。
  - `A` はタップで `A`、ホールドで `Shift`。
  - `F` (Layer2)、`G` (Layer4)、`V` (Layer3) はホールドで各レイヤーを切り替えます。
- `BS_DEL` カスタムキー:
  - 単押し: `Backspace`
  - `Shift` を押しながら: `Delete`
- そのほかのキー配置・コンボは従来の `user` キーマップを継承しています。

この README は、PDF と QMK ソースの差異を追跡するためのメモです。
